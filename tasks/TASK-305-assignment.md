# TASK-305 - Aplicar iconos, paleta y jerarquia visual

Equipo: Ejecucion Tecnica
Modo de ejecucion: Web Dev
Round: 43
Depende de: TASK-301
Estado: Assigned
Prioridad: P1 pre-lanzamiento

## Objetivo

Aplicar una mejora visual acotada en Web: iconos en acciones principales, paleta mas coherente para fidelizacion y jerarquia visual mas clara sin redisenar toda la app.

## Documentos a leer

- `AGENTS.md`
- `docs/MVP_RELEASE_STATUS.md`
- `docs/NEXT_PHASE_UX_COPY_POLISH.md`
- `tasks/TASK-301-HANDOFF.md`

## Alcance

Aplicar recomendaciones UX de TASK-301 sobre:

- iconos + texto en acciones importantes;
- botones primarios/secundarios;
- estados de exito/error/advertencia;
- colores de fidelizacion;
- tarjetas/paneles;
- badges de estado;
- responsive desktop/mobile.

Acciones candidatas para icono:

- Buscar cliente.
- Registrar cliente.
- Registrar compra.
- Redimir puntos.
- Activar/Renovar membresia.
- Aplicar beneficio.
- Guardar configuracion.
- Subir logo.
- Descargar CSV.
- Enviar/Reenviar invitacion.
- Aprobar/Rechazar.
- Ver detalle.
- Cerrar sesion.

## Criterios de aceptacion

- La app se siente mas clara y profesional sin parecer una landing page.
- Iconos no reemplazan texto en acciones importantes.
- No hay solapes en desktop/mobile.
- Colores tienen contraste suficiente.
- No se rompe el flujo autenticado ni rutas publicas.
- No usar iconos decorativos sin funcion.
